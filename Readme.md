## Tech Stack

- **HTML5** - Structure
- **CSS3** - Styling & responsive design
- **JavaScript** - Date operations

## How to Use

1. Open `index.html` in your browser
2. Current date info displays automatically

## How It Works

- **JavaScript**: Fetches current date with `new Date()` and formats it using `toLocaleString()`
- **CSS**: Flexbox layout with a centered card design
- **HTML**: Simple structure with DOM elements for dynamic content
